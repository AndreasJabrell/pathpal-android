# PathPal

PathPal is a group-based Android application for managing routes and real-time participants.

## Features (planned)
- Group routes
- QR-based joining
- Admin & member roles
- Real-time participant list
- Map integration
- Accessibility-first design

## Tech stack
- Kotlin
- Jetpack Compose
- Firebase (Authentication & Realtime Database)
- Google Maps (planned)
